# On-chain (Hardhat)

Simple Hardhat project with a sample contract and test.

## Prerequisites

- Node.js 24+
- pnpm 11+

## Setup

```bash
pnpm install
cd onchain
cp .env.example .env
```

## Run

```bash
pnpm build
pnpm test
```

## Local Node (optional)

```bash
pnpm hardhat node
pnpm deploy:localhost
```
